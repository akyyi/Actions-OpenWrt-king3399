# Actions-OpenWrt-king3399云编译项目



## 固件说明
利用GitHub-Actions编译king3399开发板适用的OpenWrt

跟随lean源码更新自动编译，自用稳定，默认无wifi。

> `管理ip：192.168.1.1 密码：password`

### 固件分区64+1024,分区一致可直接web升级，否则请重新写盘，首次刷入不建议保留配置，以免发生BUG。
